### Watch store
This is a Full Stack " Watch Sales Website "

## Demo
Live demo : https://react-watch-store.vercel.app/

## Built with
- React
- Hygraph (CMS)
- Toasify

## Features
- User-Friendly Interface
- Adding and removing the product you want from favorites
- Filtering
- Add product to cart
